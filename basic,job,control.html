The tasklist command displays a list of currently running processes.

It allows you to monitor the status of processes, check if certain applications or services are running

taskkill command terminates or ends a running process by its name or process ID (PID).

Can be used to stop potential malware.

Start-Job Start a new PowerShell background job.

Is useful for automating tasks and scripts that run independently from the main script

Get-Job Retrieve information about PowerShell background jobs.

When multiple background jobs are initiated, Get-Job enables you to track and manage them effectively.

Receive-Job Get the results of a background job.

When running time-consuming tasks as background jobs, Receive-Job helps in tracking the progress
