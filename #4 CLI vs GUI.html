CLI VS GUI

The performance benefits of CLI are as follows.

  • Speed you can move through the system quickly and with minimal resources.
  • You can use scripting to make task automated as well as working on multiple machines in the same network.
  • You can accomplish complex tasks with simple commands.

Some examples of things that can be done are.

  • server debugging
  • managing system logs
  • monitoring system resources

    • Using the CLI makes these process easier to do.
